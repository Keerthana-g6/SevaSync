# SevaSync: AI-Powered Decision Intelligence for Crisis Response

SevaSync is an active decision engine built for the **Google Solution Challenge**. It solves the "Coordination Chaos" NGOs face during disasters by transforming unstructured, messy field data into prioritized, predictive, and actionable response plans.

**We do not just match volunteers; we predict crises before they escalate.**

---

## 🌟 The Core Innovation

Traditional volunteer tools are static databases. SevaSync is a dynamic, intelligent engine featuring:

1. **The Prioritization Engine:** Uses Gemini AI to instantly parse messy field reports (like WhatsApp messages or radio transcripts), extract critical needs, and rank them based on an AI-calculated urgency and impact score.
2. **Predictive Crisis Forecasting:** Anticipates hidden, cascading crises and demand spikes in the next 6–12 hours (e.g., predicting a disease outbreak based on flood reports) before they happen.
3. **Trust & Transparency Layer (XAI):** Features explainable AI that generates a specific rationale for every single volunteer assignment and flags fairness/coverage gaps in underserved geographic zones.
4. **What-If Scenario Simulations:** Allows command centers to stress-test their resource allocations against hypothetical disaster escalations (e.g., "What if a flash flood cuts off Ward 4?").
5. **Live Impact Metrics:** Dynamically calculates metrics like Response Time Reduction and Volunteer Fleet Utilization.

---

## 🛠️ Google Technologies & Architecture

SevaSync leverages the Google ecosystem to provide enterprise-grade intelligence:

* **Google Gemini API (1.5 Flash):** The brain of the application. Handles deep semantic extraction, prioritization scoring, predictive forecasting, and explainable AI logic.
* **Node.js & Express:** A lightweight, high-performance backend routing requests between the client and Google's AI services.
* **Vanilla HTML/CSS/JS Frontend:** A clean, responsive dashboard designed for high-stress command center environments.
